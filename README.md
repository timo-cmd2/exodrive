# exodrive
Project EXODRIVE: A pragmatic implementation for a lane-detection AI. Base for autonomous cars


### Features:

- Lane detection AI
- Realtime RODN Radar
- Support for Car Hardware

### To be done:

- Realtime localization
- Steering control
- Lane departure warning
- Traffic light and sign detection


> Note: This project is yet in early development stage. Please do not use ExoDrive in Production since it is not yet completely finished.
